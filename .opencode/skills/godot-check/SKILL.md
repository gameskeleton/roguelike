## What I do

Just run this command on WSL or Windows (the command works in both environments) to check your Godot 4 project for errors without opening the editor:
!`cmd.exe /c "%GODOT_4_PATH% --quit --editor --headless --check-only"`

## When to use me

You can use this skill as part of your development workflow to catch issues early.
When you want to ensure that your Godot 4 project has no script errors before running or deploying it.
